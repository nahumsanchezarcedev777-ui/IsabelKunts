# Instrucciones de Instalación:
# 1. PYTHON: Usa Python 3.9 o superior.
# 2. ENTORNO VIRTUAL: ¡SIEMPRE!
#    python -m venv venv
#    Windows: venv\Scripts\activate
#    Linux/macOS/Termux: source venv/bin/activate
# 3. DEPENDENCIAS CRÍTICAS DE SISTEMA/MANUALES (INSTALAR ANTES):
#    - FFmpeg: Necesario para `openai-whisper`. Descargar de ffmpeg.org y AÑADIR AL PATH del sistema.
#              Verificar con: ffmpeg -version
#    - PyAudio: Para `SpeechRecognition` con micrófono. Instalación OS-dependiente:
#               Windows: Prerrequisito "Microsoft C++ Build Tools". Luego `pip install pipwin` y `pipwin install pyaudio` (o intenta `pip install pyaudio` directamente).
#               Linux (Debian/Ubuntu): `sudo apt-get update && sudo apt-get install python3-pyaudio portaudio19-dev libasound2-dev`
#               Linux (Fedora): `sudo dnf install python3-pyaudio portaudio-devel alsa-lib-devel gcc redhat-rpm-config`
#               macOS: `brew install portaudio && pip install pyaudio`
#    - PyTorch: Para `transformers`, `sentence-transformers`, `openai-whisper`. Instalar DESDE pytorch.org para la versión correcta (CPU o CUDA GPU).
#               Ej. CPU: pip3 install torch torchvision torchaudio
#               Ej. CUDA 12.1: pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
# 4. INSTALACIÓN GENERAL (con entorno virtual activo):
#    pip install -r requirements.txt
# 5. DESCARGA DATOS NLTK:
#    python -c "import nltk; nltk.download('vader_lexicon'); nltk.download('punkt')"
# 6. (SOLO WINDOWS): Si `pyttsx3` falla después de todo, intenta: `pip install pypiwin32`
